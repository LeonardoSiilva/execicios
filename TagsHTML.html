O que sao Tags Dento do HTML?

Resp: Tags sao elementos que dentro do HTML quer dizer qual o tipo de estrutura esta sendo construida

Tipos de TAGs

  • Titulo
  • Paragrafo
  • Marcaçoes
  • Tipografia
  • listas
  • HyperLinks
  • < SUB> e < SUP>
  • Anotaçoes
  • Fonte
  • del
  • tag de abreviação

Titulo.

As tags < h1 > ate o < h6 > sao tags de titulos, conforme o numero muda o tamanho da fonte, sendo a prioridade do 1 ate o 6.

Paragrafo

A tag < p > e a tag ultilizada para poder escrever textes, sendo esses menoque que os Titulos

Marcação

Para marcar texto usamos a tag < mark > selecionamos a palavra com a borda amarela Exemplo: Bom dia!

Tipografia

Tipografia sao tags para alterar a forma de uma letra,

  • < strong>. Serve para deixar a letra em negrito. Exemplo: Ola

  • < i>. Serve para deixar a letra em italico. Exemplo: Ola
  • < u>. Serve para colocar um Traço abaixo da letra Exemplo: Ola

Listas de Ordens

listas < ol> Ordenadas e < ul> nao Ordenadas. sao ultilizadas para colocar topicos Exemplo:
Listas Ordenadas sao exibidas na foma de Numeros: 1, 2 ,3 ,4 e etc.
listas nao Ordenadas elas sao feitas com uma bolinha no lugar do numero

HyperLinks

No HTML, os links são definidos pela tag < a>. Dentro dessa tag incluímos o atributo href (Hypertext Reference), que é o endereço de destino do link.

SUB e SUP

O elemento SUB Este é um elemento inline destinado a marcar fragmentos de texto que devam ser renderizados em alinhamento vertical mais baixo em relação ao texto principal. Exemplo:

A fórmula química do ácido sulfúrico é H2SO4.

O elemento SUP Este é um elemento inline destinado a marcar fragmentos de texto que devam ser renderizados em alinhamento vertical mais alto em relação ao texto principal. Exemplo:

A Sra. Maria chegará na 2a. feira.

Anotaçoes

< blockquote> indica que o texto incluído é uma longa citação.

Font

Como alterar a fonte, tamanho e cor de textos em um site HTML
Para mudar a cor de um determinado texto, devemos colocar tal trecho entre as tags < font> e < /font>, porém, devemos adicionar o atributo color, que vai definir a cor que você queira, baseado na Tabela de Cores.

Del

< del> epresenta uma parte do texto que foi excluída de um documento.

Abreviaçao

< abbr> epresenta uma abreviação e opcionalmente fornece uma descrição completa para ela
